This is my personal website that scores a perfect **100** on Chrome Lighthouse audit on both [desktop](https://developers.google.com/speed/pagespeed/insights/?url=https%3A%2F%2Fsaikat.dev%2F&tab=desktop) and [mobile](https://developers.google.com/speed/pagespeed/insights/?url=https%3A%2F%2Fsaikat.dev%2F&tab=mobile). I primarily document things that I learn by means of blog posts, cheatsheets and reference docs here. They are searchable on the blog posts list page. I also summarize and break down projects that I've built over here. This also acts as a portfolio of sorts

## Highlights of the website include:
- Perfect Lighthouse score of **100** on both **desktop** and **mobile**
- **Completely prerendered** and served from Cloudflare CDNs for blazing fast speeds
- Implements **prefetch** so that links hovered on prefetch content required to load page on navigation
- Includes a strictly **clientside search** on the "all blogs" listing using **lunr** that loads the **search index on demand**
- Styled using a neon theme that draws inspiration from Cyberpunk 2077
- Includes video walkthroughs for projects that have been retired for a quick walkthrough
